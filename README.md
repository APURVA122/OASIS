# OIBSIP
# 🖥️ Apurva OS 98

## 🌐 Live Demo
https://apurva122.github.io/OIBSIP/

A retro Windows 98-inspired portfolio and project hub built using HTML, CSS, and JavaScript.

This project recreates the look and feel of an old desktop computer, allowing visitors to explore projects through a nostalgic Windows-style interface.

## ✨ Features

* 🖥️ Windows 98-inspired desktop UI
* 📁 Project Explorer window
* 📄 About Me window
* 🕒 Live taskbar clock
* 🎨 Classic retro styling
* 📱 Responsive design
* 🔗 Direct access to individual projects

## 🚀 Projects Included

* 🧮 Calculator
* 📝 To-Do App
* 🔐 Login Authentication System
* 🇮🇳 Tribute Page

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
